# Get Minecraft For Free!!! (In Chinese)

今天我们来学习白嫖Minecraft。

首先，[点击这里](https://www.oracle.com/cn/java/technologies/downloads/)，下载JDK安装包并运行。

![JDK官网](install_java.png)

然后，[点击这里](https://github.com/HMCL-dev/HMCL)，下载HMCL。

![HMCL下载](hmcl.png)

最后运行，添加一个离线账号，下载游戏并启动。

关于Minecraft的问题，请前往 [https://www.mcmod.cn/](https://www.mcmod.cn/) 。
